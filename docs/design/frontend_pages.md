```

/images                     # 镜像管理

/training                   # 训练任务

/training/:id               # 任务详情
/inference                  # 推理服务
/artifacts                  # 制品管理
/monitoring                 # 监控面板
```

## 2. 用户端页面

### 2.1 认证相关页面

#### 2.1.1 登录页 `/login`

**页面功能：** 用户登录

**表单字段：**
- 用户名/邮箱（必填）
- 密码（必填）
- 记住我（可选）

**其他元素：**
- 忘记密码链接
- 注册账号链接
- OAuth登录按钮（GitHub、Google）
- LDAP登录选项

**API调用：**
- `POST /api/auth/login` - 用户名密码登录
- `GET /api/auth/oauth/:provider` - OAuth登录

---

#### 2.1.2 注册页 `/register`

**页面功能：** 用户注册

**表单字段：**
- 用户名（必填，3-20字符）
- 邮箱（必填，需验证）
- 密码（必填，8-32字符）
- 确认密码（必填）
- 验证码（必填）
- 同意服务条款（必填）

**API调用：**
- `POST /api/auth/register` - 注册
- `POST /api/auth/send-verification-code` - 发送验证码

---

### 2.2 概览页面

#### 2.2.1 仪表板 `/dashboard`

**页面功能：** 显示用户资源使用概览

**数据卡片：**
- 运行中的环境数量
- 总GPU使用时长
- 本月费用
- 存储使用量

**图表：**
- 资源使用趋势（折线图）
- GPU使用率分布（饼图）
- 最近7天费用（柱状图）

**列表：**
- 最近使用的环境（表格）
  - 列：环境名称、状态、GPU、创建时间、操作
- 最近的训练任务（表格）
  - 列：任务名称、状态、进度、开始时间

**API调用：**
- `GET /api/dashboard/overview` - 获取概览数据
- `GET /api/dashboard/resource-usage` - 资源使用趋势
- `GET /api/environments?limit=5` - 最近环境
- `GET /api/training/jobs?limit=5` - 最近任务

### 2.3 环境管理页面

#### 2.3.1 环境列表 `/environments`

**页面功能：** 显示和管理所有开发环境

**筛选器：**
- 状态（全部、运行中、已停止、错误）
- 镜像类型（全部、PyTorch、TensorFlow、CUDA）
- 创建时间范围

**搜索框：**
- 按环境名称搜索

**操作按钮：**
- 创建环境（主按钮）
- 批量操作（停止、删除）

**表格列：**
- 环境名称
- 状态（运行中/已停止/错误）
- 镜像
- GPU配置（型号、数量）
- CPU/内存
- 运行时长
- 创建时间
- 操作（启动、停止、连接、删除）

**API调用：**
- `GET /api/environments` - 获取环境列表
- `POST /api/environments/:id/start` - 启动环境
- `POST /api/environments/:id/stop` - 停止环境
- `DELETE /api/environments/:id` - 删除环境

---

#### 2.3.2 创建环境 `/environments/create`

**页面功能：** 创建新的开发环境

**表单步骤：**

**步骤1：基本信息**
- 环境名称（必填）
- 描述（可选）
- 操作系统（Linux/Windows）

**步骤2：选择镜像**
- 镜像类型（官方镜像/自定义镜像）
- 镜像列表（卡片展示）
  - PyTorch 2.0 + CUDA 11.8
  - TensorFlow 2.13 + CUDA 11.8
  - CUDA 12.0 Base
  - 自定义镜像...

**步骤3：资源配置**
- CPU核心数（滑块，1-32）
- 内存大小（滑块，2GB-128GB）
- GPU数量（滑块，0-8）
- GPU型号（下拉选择）
- 存储空间（滑块，10GB-500GB）

**步骤4：网络配置**
- SSH访问（开关）
- RDP访问（开关，仅Windows）
- JupyterLab（开关）
- 自定义端口（可选）

**步骤5：数据挂载**
- 选择数据集（多选）
- 选择模型（多选）

**步骤6：确认创建**
- 显示配置摘要
- 预估费用
- 创建按钮

**API调用：**
- `GET /api/images` - 获取镜像列表
- `GET /api/datasets` - 获取数据集列表
- `GET /api/scheduler/available-resources` - 查询可用资源
- `POST /api/environments` - 创建环境

---

#### 2.3.3 环境详情 `/environments/:id`

**页面功能：** 查看和管理单个环境

**顶部信息卡：**
- 环境名称
- 状态标签
- 操作按钮（启动、停止、重启、删除）

**标签页：**

**Tab 1: 概览**
- 基本信息
  - 环境ID
  - 创建时间
  - 运行时长
  - 镜像信息
- 资源配置
  - CPU、内存、GPU
  - 存储使用情况
- 访问信息
  - SSH连接命令
  - RDP连接地址
  - JupyterLab链接

**Tab 2: 监控**
- 实时监控图表
  - CPU使用率（折线图）
  - 内存使用率（折线图）
  - GPU使用率（折线图）
  - GPU显存（折线图）
  - 网络流量（折线图）

**Tab 3: 终端**
- Web终端（xterm.js）
- 支持SSH连接

**Tab 4: 文件管理**
- 文件浏览器
- 上传/下载文件
- 创建/删除文件夹

**Tab 5: 日志**
- 环境日志（实时滚动）
- 日志级别筛选
- 日志搜索

**Tab 6: 设置**
- 修改环境名称
- 修改描述
- 自动停止设置
- 删除环境

**API调用：**
- `GET /api/environments/:id` - 获取环境详情
- `GET /api/monitoring/environments/:id/metrics` - 获取监控数据
- `GET /api/environments/:id/logs` - 获取日志
- `PUT /api/environments/:id` - 更新环境
