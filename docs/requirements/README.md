## 已创建的需求文档

### 模块 1：CMDB 设备管理模块

1. **[主机注册与发现](./01-主机注册与发现.md)** (P0)
   - 自动发现主机（Agent 心跳）
   - 手动注册主机（SSH 连接）
   - 主机信息采集（CPU、内存、磁盘）
   - 操作系统识别（Linux/Windows）
   - 部署模式识别（Docker/Kubernetes）

2. **[GPU 设备管理](./02-GPU设备管理.md)** (P0)
   - GPU 自动发现（nvidia-smi）
   - GPU 信息采集（型号、显存、UUID）
   - GPU 分配与释放
   - GPU 健康检查（温度、功耗、ECC 错误）
   - GPU 拓扑关系管理（NVLink、PCIe）

3. **[设备状态管理](./03-设备状态管理.md)** (P0)
   - 运营状态管理（available、maintenance、faulty、retired、reserved）
   - 使用状态管理（idle、partial、full、overcommit）
   - 健康状态监测（healthy、degraded、unhealthy）
   - 状态转换控制（状态机、前置条件检查）
   - 自动隔离机制

### 模块 2：用户与权限模块

4. **[用户认证与授权](./04-用户认证与授权.md)** (P0)
   - 用户名密码认证
   - OAuth2 认证（GitHub、Google、企业微信）
   - LDAP/AD 认证
   - JWT Token 管理
   - RBAC 权限模型

12. **[工作空间管理](./12-工作空间管理.md)** (P1)
   - 工作空间类型（personal、team、enterprise）
   - 成员角色管理（owner、admin、member、viewer）
   - 邀请成员加入
   - 资源共享和权限隔离

13. **[配额管理](./13-配额管理.md)** (P1)
   - 资源配额设置（CPU、内存、GPU、存储）
   - 配额级别（Free、Basic、Pro、Enterprise）
   - 配额检查和使用量统计
   - 配额升级

### 模块 3：环境管理模块

5. **[开发环境创建与管理](./05-开发环境创建与管理.md)** (P0)
   - 环境创建流程
   - 支持多种镜像（PyTorch、TensorFlow、CUDA）
   - 资源配置（CPU、内存、GPU）
   - SSH/RDP 访问
   - JupyterLab 集成

14. **[JupyterLab 集成](./14-JupyterLab集成.md)** (P1)
   - JupyterLab 启动和停止
   - Token 认证和访问控制
   - 反向代理配置
   - 健康检查

### 模块 4：任务调度模块

6. **[资源调度策略](./06-资源调度策略.md)** (P0)
   - 首次适配（First Fit）
   - 最佳适配（Best Fit）
   - 最少负载（Least Loaded）
   - 轮询（Round Robin）
   - 并发调度控制

7. **[端口管理](./07-端口管理.md)** (P0)
   - 端口池管理
   - 动态端口分配
   - 端口映射（SSH、RDP、JupyterLab）
   - 端口回收机制

15. **[离线训练任务](./15-离线训练任务.md)** (P1)
   - 任务提交和调度
   - 任务生命周期管理
   - 任务监控和日志查看
   - 失败重试机制

### 模块 5：网络访问模块
