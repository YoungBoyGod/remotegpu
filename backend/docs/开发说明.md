# 开发说明

## 热更新启动

需要先安装 `air`：

```
go install github.com/cosmtrek/air@latest
```

启动热更新：

```
make dev
```

也可以直接运行：

```
air
```

## Swagger/OpenAPI

- Swagger UI: `http://<host>:<port>/swagger/index.html`
- OpenAPI 文档: `http://<host>:<port>/openapi.yaml`
- OpenAPI 路径可通过 `swagger.openapi_path` 配置
- 避免设置为 `/swagger/openapi.yaml`，会与 `/swagger/*any` 路由冲突

## 日志展示域名

`server.host` 仅用于启动日志里展示访问地址：

```
server:
  host: "localhost"      # 或示例域名
  # host: "api.example.com:8080"  # 包含端口时可直接写完整地址
```

## 管理员重置

重置管理员密码（默认用户为 `admin`）：

```
./bin/remotegpu admin reset --user admin --password 新密码
```
