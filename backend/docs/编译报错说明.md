# 编译报错说明

## 报错信息

执行 `make build` 时出现：

```
cmd/main.go:4:2: undefined: Execute
```

## 原因

`Makefile` 中的编译命令是 `go build -o bin/remotegpu cmd/main.go`，只编译了单个文件。
Go 在只给定单个文件时，不会自动把同目录下的其他文件（如 `cmd/root.go`）一起编译，
而 `Execute()` 就定义在 `cmd/root.go` 中，因此出现未定义错误。

## 解决方案

改为编译整个 `cmd` 包，让 Go 读取该目录下所有文件：

- `go build -o bin/remotegpu ./cmd`
- `go run ./cmd`

本次已在 `Makefile` 中同步修改上述命令，重新执行 `make build` 即可通过编译。
